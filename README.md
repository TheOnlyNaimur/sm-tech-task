# Floka Studio — Modern Portfolio & Creative Agency Website

A high-performance, fully responsive React portfolio website inspired by Floka's design system. Built with cutting-edge web technologies including Three.js WebGL shaders, Tailwind CSS v4, and Vite for optimal performance.

## ✨ Features

- **Responsive Design** — Mobile-first approach with seamless scaling from 320px to 4K displays
- **Splash Screen** — Black loading screen with animated white linear bar on initial load
- **WebGL Image Effects** — FunFacts ripple shader, Awards diamond grid distortion
- **Smooth Animations** — Scroll-reveal, parallax hover, staggered entrance animations
- **Component Library** — 18+ reusable React components
- **Performance Optimized** — Vite build with code splitting
- **SEO Ready** — Semantic HTML, meta tags

## 🏗️ Tech Stack

- **React 19** + **TypeScript 5.9**
- **Vite 8** — Lightning-fast builds
- **Tailwind CSS v4** — Utility-first styling
- **Three.js r150** — WebGL shaders
- **GSAP** + **Lenis** — Smooth animations
